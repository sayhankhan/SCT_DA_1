# Superstore Sales Dashboard using Power BI

## Project Overview

This project analyzes retail sales data using Power BI. The dashboard provides insights into sales performance, profitability, and sales trends over time.

## Features

* Total Sales KPI
* Total Profit KPI
* Sales by Category Analysis
* Profit by Category Analysis
* Sales Trend Over Time
* Interactive Filters (Category, Region, Year)

## Tools Used

* Power BI
* Power Query
* Data Visualization
* Business Intelligence

## Dataset

Superstore Sales Dataset

## Key Insights

* Technology generated the highest sales and profit.
* Sales performance varies across categories.
* Time-series analysis helps identify sales trends and seasonality.

## Outcome

Built an interactive business dashboard to support data-driven decision-making.
